let和const

基本使用

ES5中声明变量都是使用var关键字，从ES6开始新增了let和const两个关键字来声明变量。

let和var没有太大区别；const用于声明常量，它保存的数据一旦被赋值，就不能被修改。

let、const不允许重复声明变量。

作用域提升

在声明变量的作用域中，如果这个变量可以在声明之前被访问，那么可以称为作用域提升

var声明的变量会进行作用域提升，但let/const声明的变量在声明之前访问会报错。

从块作用域顶部一直到变量声明完成之前，这个变量处在暂时性死区。在执行上下文的词法环境创建出来的时候，变量实际上已经被创建了，只是这个变量是不能被访问的。

在全局通过var来声明一个变量，事实上会在window上添加一个属性，而let、const不会在window上添加任何属性。

块级作用域

在ES5中，javaScript只会形成两个作用域：全局作用域和函数作用域。在代码块中定义的变量可以在外部访问。

ES6新增了块级作用域，通过let、const、function、class声明的标识符是具备块级作用域的限制的。

函数拥有块级作用域，但是外面依然可以访问，这是因为引擎会对函数的声明进行特殊处理。

字符串模板

ES6允许使用字符串模板来嵌入JS的变量或表达式来进行拼接。
使用``符号来编写字符串，称为模板字符串，在模板字符串中，可以通过${expression}来嵌入动态的内容。

模板字符串还有种用法：标签模板字符串。
如果使用标签模板字符串，并在调用的时候插入其他变量：

  • 模板字符串被拆分
  • 第一个元素是数组，是被模块字符串拆分的字符串组合
  • 后面的元素是一个个模块字符串传入的内容

默认值会改变参数length的个数，默认值以及后面的参数不计算在length内。

函数的剩余参数

ES6中引用了rest parameter，可以将不定数量的参数放到一个数组中。
如果最后一个参数是…为前缀的，那么它会将剩余的参数放到该参数中，并作为一个数组。

剩余参数和arguments的区别：

  • 剩余参数只包含没有对应形参的实参，而argments对象包含了传给函数的所有实参
  • arguments对象不是一个真正的数组，而rest参数是一个真正的数组
  • arguments是早期的ECMAScript中为了方便去获取所偶的参数提供的一个数据结构，而rest参数是ES6中提供并希望以此替代arguments的

注意：剩余参数必须放到最后一个位置，否则会报错。

箭头函数

箭头函数没有显示原型prototype，所以不能作为构造函数，使用new来创建对象。
箭头函数不绑定this、arguments、super参数。

Symbol

Symbol是ES6中新增的一个基本数据类型，译为符号。
在ES6之前，对象的属性名都是字符串形式，很容易造成属性名的冲突。在给原有对象添加新属性时，可能会覆盖之前的属性。
Symbol就是为了解决这个问题，它能生成一个独一无二的值：

  • Symbol值通过Symbol函数生成，生成后可以作为属性名
  • 在ES6中，对象的属性名可以使用Symbol值
  • Symbol即使多次创建值，它们也是不同的
  • 可以在创建Symbol值的时候传入一个描述（ES10新特性）

可以使用Symbol.for方法创建相同的Symbol，可以通过Symbol.keyFor方法获取对应的key。

Set

基本使用

在ES6之前，存储数据的解构主要有两种：数组、对象。在ES6中新增了另外两种数据解构：Set、Map，以及它们的另外形式WeakSet、WeakMap。
Set是一个新增的数据结构，可以用来保存数据，类似于数组，但元素不能重复，可用来给数组去重。
创建Set需要通过Set构造函数。

Set的常见方法

Set常见属性：size，返回Set中元素的个数
Set常用方法：

  • add(value)：添加某个元素，返回Set对象本身
  • delete(value)：从set中删除和这个值相等的元素，返回boolean类型
  • has(value)：判断set中是否存在某个元素，返回boolean类型
  • clear()：清空set中所有的元素，没有返回值
  • forEach(callback, [, thisArg])：通过forEach遍历set

Set支持for of的遍历。

WeakSet使用

WeakSet也是内部元素不能重复的数据结构，和Set的区别：

  • WeakSet中只能存放对象类型，不能存放基本数据类型
  • WeakSet对元素的引用是弱引用，如果没有其他引用对某个对象进行引用，那么GC可以对该对象进行回收

WeakSet常见方法：

  • add(value)：添加某个元素，返回WeakSet对象本身
  • delete(value)：从WeakSet中删除和这个值相等的元素，返回boolean类型
  • has(value)：判断WeakSet中是否存在某个元素，返回boolean类型

注意：WeakSet不能遍历，因为WeakSet只是对对象的弱引用，如果遍历获取到其中的元素，有可能造成对象不能正常销毁，所有存储到WeakSet中的对象是没法获取的。

Map

基本使用

Map用于存储映射关系。Map可以使用对象作为key，这时会自动将对象转成字符串。

Map的常用方法

Map常见属性：size，返回Map中元素的个数
Map常见方法：

  • set(key, value)：在Map中添加key、value，并返回整个Map对象
  • get(key)：根据key获取Map中的value
  • has(key)：判断是否包括某个key，返回Boolean类型
  • delete(key)：根据key删除一个键值对，返回Boolean类型
  • clear()：清空所有元素
  • forEach(callback, [, thisArg])：通过forEach遍历Map

Map也可以通过for of进行遍历。

WeakMap的使用

WeakMap也是以键值对的形式存在的，和Map的区别：

  • WeakMap的key只能使用对象，不接受其他的类型作为key
  • WeakMap的key对对象的应用是弱引用，如果没有其他引用引用这个对象，那么GC可以回收该对象

WeakMap常见的方法有四个：

  • set(key, value)：在Map中添加key、value，并且返回整个Map对象
  • get(key)：根据key获取Map中的value
  • has(key)：判断是否包括某一个key，返回Boolean类型
  • delete(key)：根据key删除一个键值对，返回Boolean类型

注意：WeakMap不能遍历。

ES10新增特性

flat

flat()方法会按照一个可指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回。

flatMap()方法首先使用映射函数映射每个元素，然后将结果压缩成一个新数组。
注意：

  • flatMap首先进行map操作，再做flat的操作
  • flatMap中的flat相当于深度为1

fromEntries

Object.entries可以将一个对象转换为entries，Object.fromEntries可以将entries转换成对象。

trimStart和trimEnd

trim会去除一个字符串首位的空格，ES10提供了trimStart和trimEnd单独去除前面或后面的空格。

其他

Symbol description
Optional catch binding

ES11新增特性

BigInt

ES11中引入了新的数据类型BigInt，用于表示大的整数。BigInt的表示方法是在数值的后面加上n。

空值合并操作符

可选链

可选链是ES11中新增的特性，主要作用是在进行null和undefined判断时更加清晰和简洁。

globalThis

之前想获取JavaScript环境的全局对象，不同环境获取方式不同，如在浏览器中可以通过this、window获取；在Node中需要通过global来获取。

在ES11中对获取全局对象进行了统一的规范：globalThis。

for..in标准化

ES11中，ECMA对for..in进行了标准化，其用于遍历对象的key。

其他

Dynamic Import
Promise.allSettled
import meta

ES12新增特性

FinalizationRegistry

FinalizationRegistry对象可以让你在对象被垃圾回收时请求一个回调。
FinalizationRegistry提供了这样一个方法：当一个在注册表中注册的对象被回收时，请求在某个时间点上调用一个清理回调。
可以通过调用register方法，注册任何想要清理回调的对象，传入该对象和所含的值。

WeakRef

如果默认将一个对象赋值给另外一个引用，那么这个引用是一个强引用。如果希望是弱引用，可以使用WeakRef。

其他

logical assignment operators:

Numeric Separator
String.replaceAll

ES13新增特性

ES13加入了字符串和数值的at方法：

Object新增了一个静态方法：hasOwn(obj, propKey)，该方法用于判断一个对象中是否有某个自己的属性。

与Object.prototype.hasOwnProperty区别：

  • 防止对象内部有重写hasOwnProperty
  • 对于隐式原型指向null的对象，hasOwnProperty无法判断

ES13新增了定义class类中成员字段的其他方式：

  • instance public fields
  • static public fields
  • instance private fields
  • static private fields
  • static block
